# Testing on Rails

By Nicolas Culerier et Nicolas Zhao. Des tests en série, pétages de câble absolus au codage...

## Prérequis

Ruby Version : 2.5.1

Rails Version : 5.2.0

## Déploiement en local

```git clone``` https://github.com/nicoc-irtanos/THP_Testing_Rails

```cd THP_Testing_Rails```

```rails db:create```

```rails db:migrate```

```rails db:seed```

```rails test```
